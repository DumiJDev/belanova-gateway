Testing Instructions

  1. Access the gateway at http://localhost:8080
  2. Try the endpoints above with the /api/ prefix
  3. Check the admin UI at http://localhost:8081
  4. Monitor metrics at http://localhost:9090
  5. View dashboards at http://localhost:3000

Rate Limiting Test

Try making multiple requests quickly to test rate limiting:

for i in {1..20}; do curl -s http://localhost:8080/api/users; done

JWT Authentication Test

The gateway includes JWT authentication. Use the admin UI to configure and test authentication.
